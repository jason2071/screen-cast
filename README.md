# Screen Cast

แอป Android ที่รวมการแคสต์ไว้ในหน้าเดียว:

- **Miracast / Screen Mirroring:** เปิดหน้าจอ Cast ของระบบเพื่อสะท้อนทั้งหน้าจอไปยังทีวีที่รองรับ
- **Google Cast / Chromecast:** ค้นหา Chromecast หรือ Google TV และส่งวิดีโอ MP4, HLS (`.m3u8`) หรือ DASH (`.mpd`) จาก URL ไปเล่นบนทีวี
- **วิดีโอในเครื่อง:** เลือกไฟล์วิดีโอในโทรศัพท์แล้วส่งขึ้นทีวีผ่าน Google Cast โดยไม่ต้องอัปโหลดที่ไหน

## การทำงาน

Miracast ถูกควบคุมโดย Android และผู้ผลิตอุปกรณ์ แอปจึงเปิดหน้าจอ Cast ของระบบโดยตรง ซึ่งเป็นช่องทางที่เข้ากันได้ที่สุดสำหรับโทรศัพท์แต่ละรุ่น

ส่วน Google Cast ใช้ **Default Media Receiver** ของ Google จึงใช้งานกับ URL สื่อสาธารณะได้ทันทีโดยไม่ต้องสมัคร Receiver app ID เพิ่มเติม แต่ URL ต้องให้ Chromecast/Google TV เข้าถึงได้จากเครือข่าย และวิดีโอต้องรองรับรูปแบบที่เลือก

### วิดีโอในเครื่อง

Chromecast ดึงสื่อเองผ่านเครือข่าย จึงเข้าถึง `content://` หรือพาธในโทรศัพท์ไม่ได้ แอปจึงเปิด HTTP server ชั่วคราวบนที่อยู่ Wi‑Fi ของเครื่อง (`LocalMediaServer`) แล้วส่ง URL นั้นให้ทีวีดึงไฟล์ไปเล่น

- เลือกไฟล์ผ่าน Storage Access Framework จึงไม่ต้องขอสิทธิ์เข้าถึงที่เก็บข้อมูล
- รองรับ HTTP Range request ทีวีจึงเลื่อนตำแหน่งเล่นได้
- แต่ละไฟล์ได้พาธสุ่ม เครื่องอื่นในวง Wi‑Fi เดาไม่ได้
- เปิดหน้าจอค้างไว้ระหว่างส่ง เพราะเซิร์ฟเวอร์อยู่ในอายุของ Activity

## ดาวน์โหลด APK

หน้า [Releases](https://github.com/jason2071/screen-cast/releases) มีไฟล์ APK ของทุกเวอร์ชันที่ติดแท็กไว้ ดาวน์โหลดได้โดยไม่ต้องล็อกอิน

ออกเวอร์ชันใหม่โดยติดแท็กแล้วผลักขึ้นต้นทาง GitHub Actions จะ build และสร้าง release ให้เอง:

```bash
git tag v1.0.0
git push origin v1.0.0
```

APK จาก workflow `Android CI` ของทุก commit ก็โหลดได้จากหน้า run นั้น แต่ต้องล็อกอิน และไฟล์หมดอายุใน 90 วัน

## เปิดใน Android Studio

1. เปิดโฟลเดอร์นี้ด้วย Android Studio
2. ตั้งค่า Android SDK Platform 35 และ JDK 17 หรือใหม่กว่า
3. รอ Gradle sync แล้วเลือกอุปกรณ์ Android ที่ใช้ API 26 ขึ้นไป
4. กด Run

หากเครื่องไม่มี JDK 17 Gradle จะดาวน์โหลดให้เองผ่าน foojay toolchain resolver ที่ตั้งไว้ใน `settings.gradle.kts`

## ข้อจำกัดที่ตั้งใจออกแบบไว้

- โทรศัพท์และทีวีต้องรองรับ Miracast/Screen Mirroring สำหรับการสะท้อนทั้งหน้าจอ
- Google Cast ไม่ใช่การสะท้อนทั้งหน้าจอ: ใช้สำหรับส่งสื่อไปยัง Chromecast หรือ Google TV โดยตรง
- บางรุ่นปิดหรือเปลี่ยนตำแหน่งเมนู Cast โดยผู้ผลิต แอปจะพาไปหน้า Settings แทนหากเรียกเมนู Cast ไม่ได้
- หน้าจอที่มี DRM เช่นบริการสตรีมมิงบางราย อาจไม่อนุญาตให้สะท้อนภาพ
- การส่งไฟล์ในเครื่องต้องอยู่บน Wi‑Fi วงเดียวกับทีวี และไม่มีการแปลงรูปแบบไฟล์ ทีวีจึงต้องรองรับ codec ของไฟล์นั้นเอง
- ไฟล์บนคลาวด์ที่ยังไม่ได้ทำ offline อาจเปิดอ่านขนาดไม่ได้ และจะส่งไม่สำเร็จ
